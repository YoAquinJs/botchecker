### Configuration
Create `.env` or `config.json` file

####_Json_
```json
{
  "token": "bot_token",
  "botId": "bot_client_id",
  "guildId": "target_user_guild_id",
  "targetId": "target_id",
  "fileName": "initial_file"
}

```
####_Env_
```dotenv
token="bot_token"
botId="bot_client_id"
guildId="target_user_guild_id"
targetId="target_id"
fileName="initial_file"
```
